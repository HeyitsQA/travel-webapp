# Travel Logbook

## Projektbeschreibung

Diese Web-App ermöglicht es Nutzern, Orte auf den Trips (z. B. Restaurants, Sehenswürdigkeiten, Cafés) zu speichern und zu verwalten – sowohl besuchte als auch geplante. Zu jedem Ort können persönliche Notizen, Bewertungen, Preise sowie genaue Standorte (inkl. Google Maps Integration) hinterlegt werden. Zusätzlich unterstützt die App beim Entdecken neuer Orte durch Empfehlungen.

## Tech Stack

* **Java & Spring Boot** – Backend & API
* **Gradle** – Build-Tool
* **Vue.js** – Frontend
* **Render** – Deployment
* **Externe APIs** (z. B. Google Maps)

## Projektziele

* Zentrale Plattform zur Organisation persönlicher Orte und Erlebnisse
* Kombination aus Tracking (Vergangenheit) und Planung (Zukunft)
* Intuitive UI zur schnellen Erfassung von Notizen, Bewertungen und Kosten
* Integration externer Datenquellen für bessere Standortinformationen

## Geplante Funktionen

1. Benutzerregistrierung und -authentifizierung
2. Orte speichern (besucht / geplant) mit Status-Tracking
3. Integration von Google Maps für genaue Standorte und Anzeige
4. Persönliche Notizen und Bewertungen pro Ort
5. Tracking von Ausgaben (z. B. Essen, Unterkunft) auf die Reise
6. Personalisierte Ortsempfehlungen basierend auf die Vorliebe des Nutzer
7. Filter- und Suchfunktionen (z. B. nach Preis, Kategorie, Bewertung)
